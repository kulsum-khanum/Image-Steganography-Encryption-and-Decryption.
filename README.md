# Image Steganography - Encryption and Decryption

A secure, Django-based web application designed to hide sensitive information within images using Least Significant Bit (LSB) steganography techniques.

## 🚀 Overview
This application provides a robust platform for data privacy, allowing users to securely encrypt messages and embed them into images. Developed as a core project during my academic internship, it demonstrates practical implementation of cybersecurity principles and Python web development.

## 🛠️ Getting Started
1. Clone the repository: `git clone https://github.com/kulsum-khanum/Image-Steganography-Encryption-and-Decryption.git`
2. Install requirements: `pip install -r requirements.txt`
3. Run migrations: `python manage.py migrate`
4. Start the server: `python manage.py runserver`
